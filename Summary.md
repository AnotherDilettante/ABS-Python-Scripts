### **Project: ABS Data Automation & Visualization Pipeline**

**Goal:** Automate the extraction, cleaning, and visualization of key Australian economic indicators (GDP, CPI, Labour Force, etc.) from the Australian Bureau of Statistics (ABS).

**Tech Stack:** Python, Pandas, Matplotlib, `readabs` (package).

---

### **1. Script Directory**

#### **`s00_readabs_datalist.py` (The Configuration)**
*   **Function:** Acts as the "Master Registry" or "Shopping List" for the project.
*   **Key Features:**
    *   Contains a dictionary list (`ABS_DATASETS`) defining the Catalogue ID, Frequency, and Tables to fetch.
    *   Supports two modes:
        1.  **Simple:** Download a table and save the default view.
        2.  **Advanced:** Download a table but filter specifically for a list of **Series IDs** (e.g., `A2304402X`).
    *   Sets global variables like `HISTORY_YEARS` (currently 10 years).

#### **`s01_readabs_datadownload.py` (The Extract & Load)**
*   **Function:** Iterates through `s00` and downloads data.
*   **Key Features:**
    *   Uses `readabs` to fetch the full catalogue from the ABS website.
    *   Calculates rows to keep based on frequency (Monthly=120 rows, Quarterly=40 rows).
    *   **Filtering:** If "Advanced" mode is used in `s00`, it strips out all columns *except* the requested Series IDs before saving.
    *   **Output:** Saves clean CSV files to `abs_data_output/`.

#### **`s02_readabs_plotting.py` (The Visualizer)**
*   **Function:** Loops through the CSV files generated by `s01` and generates standardized line charts.
*   **Key Features:**
    *   **Styling:** Uses a custom style (Red/Black lines, clean spines, "Roboto" font with "Arial" fallback).
    *   **Dynamic:** Plots specific columns if defined in `s00`, otherwise defaults to the headline column.
    *   **Output:** Saves PNG images to `abs_charts_output/`.

#### **`s03_construction_cycle.py` (Specific Analysis)**
*   **Function:** A specialized script for cross-dataset analysis (Building Cycle).
*   **Key Features:**
    *   Demonstrates how to load **Monthly** data (Approvals) and **Quarterly** data (Commencements).
    *   Uses Pandas `.resample()` to sum Monthly data into Quarterly figures for valid comparisons.
    *   Plots multiple datasets on a single chart.

---

### **2. Project Outputs**

*   **`abs_data_output/`**: Contains the raw or filtered CSV data files (ignored by Git).
*   **`abs_charts_output/`**: Contains the generated PNG charts (ignored by Git).
*   **`.readabs_cache/`**: Hidden folder where `readabs` stores raw Excel downloads to speed up subsequent runs (ignored by Git).

1.  **Populate Series IDs:**
    *   You have set up the structure in `s00`, but many datasets (Labour Force, Balance of Payments, etc.) still need specific **Series IDs** added to filter the data down to just the useful lines.

2.  **Calculations (YoY / QoQ):**
    *   Currently, `s02` plots the **Raw Index/Value**.

3.  **The "Red Light" Files (Manual Scraper):**
    *   `readabs` cannot handle non-standard files like **Overseas Migration (3407.0)** or **IIP Supplementary (5352.0)** because they are Data Cubes or Zip files.

4.  **Command Line Arguments (CLI):**
